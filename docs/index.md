## Getting Started

### Build
```bash
cmake .
make clean all
```

### Run
```bash
./beastem                    # Default firmware
./beastem -f myrom.bin       # Custom ROM
./beastem -d2 videobeast.dat # With VideoBeast
```

### Debug
- Press `ESC` to enter debug view
- `R` to run, `S` to step, `Q` to quit

## For AI-Assisted Development

When working on this codebase:

1. **Entry point**: Start at `beastem.cpp:main()` for initialization flow
2. **Emulation core**: `Beast` class in `src/beast.cpp` coordinates all components
3. **Adding peripherals**: Implement `I2cDevice` interface and register with I2C bus
4. **Memory operations**: See `Beast::readMem()` / `Beast::writeMem()` for paging logic
5. **Video rendering**: `VideoBeast::tick()` handles frame rendering cycle
